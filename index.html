Installation

Add this line to your application's Gemfile:

gem 'echonest-ruby-api'

And then execute:

$ bundle

Or install it yourself as:

$ gem install echonest-ruby-api

Usage

Require the gem in your file:

require 'echonest-ruby-api'

Artist

Create an instance of an object

artist = Echonest::Artist.new('Weezer', 'YOUR-API-KEY')

Then you have access to a bunch of methods:

artist.name
artist.biographies
artist.blogs
artist.familiarity
artist.hotttnesss
artist.images
artist.songs

Song

Create an instance of the Song module.

song = Echonest::Song.new('YOUR-API-KEY')

Then you have access to the song/search endpoint: (this is where it gets clever)

params = { mood: "sad^.5", results: 10, min_tempo: 130, max_tempo: 150 }
song.search(params)

See the full list of params here

You can even identify a song simply from its fingerprint!

Firstly generate an Echoprint code. You'll need to run and compile this for whatever platform you're using. echonest/echoprint-codegen Then just pass the generated code to the song.identify method.

song.idenfity(code)

Checkout spec/song_spec.rb for an example code.
